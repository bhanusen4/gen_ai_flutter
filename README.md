# genaiflutter

A Flutter project showcasing how to build GenAI powered applications using Firebase AI and Gemini.

---

# Getting Started

This project is a starting point for building AI-powered Flutter applications using Firebase and Gemini AI.

## Firebase Setup

Before running the project, make sure you complete the Firebase setup correctly.

### 1. Create a Firebase Project

Go to Firebase Console and create a new project.

### 2. Enable Gemini AI

Inside Firebase Console:

- Open your project
- Go to **AI Logic**
- Enable **Gemini AI**
- Configure Gemini for your Firebase project

### 3. Add Firebase to Flutter

Run:

flutterfire configure
